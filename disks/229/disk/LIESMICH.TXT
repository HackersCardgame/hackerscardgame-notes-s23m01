Was ist Total Commander?

Total Commander (früher Wincmd) ist ein Dateimanager fuer Windows aehnlich wie der
Windows-eigene Dateimanager Explorer bzw. WINFILE.EXE. Total Commander hat jedoch
2 Dateifenster nebeneinander, das das Kopieren von Dateien viel einfacher macht.
Total Commander gibt es in 2 Versionen: 16 und 32 bit. Waehrend die 16-bit Version
problemlos auch im 32-bit Windows laeuft, kann die 32-bit Version nicht unter
Windows 3.1 verwendet werden.

Total Commander ist Shareware. Die Shareware-Version darf frei verbreitet
werden, sofern sie unveraendert bleibt!

Hier einige Eigenschaften (Features) von Total Commander:

- Lange Dateinamen unter Windows 95/98/ME und Windows NT/2000/XP
- FTP-Zugang zum Laden von Dateien über das Internet
- Direkter Zugriff auf die Netzwerkumgebung (ohne ein Laufwerk verbinden zu müssen)
- Liste bereits besuchter Verzeichnisse, erlaubt zurückzugehen
- Verzeichnismenü (konfigurierbare Liste oft benutzer Verzeichnisse) - STRG+D
- Aufspalten/Zusammenfügen von grossen Dateien
- Codieren/Decodieren von Binärdateien ins UUE, XXE und MIME-Format (Text)
- Drag & Drop (Ziehen und Fallenlassen) mit der Maus.
- Drag&Drop vom und zum Explorer und Desktop
- Erweitertes Kopieren, Verschieben, Umbenennen und Loeschen ganzer Verzeichnis-
  baeume auf einmal, mit Sicherheitsabfrage.
- Gepackte Dateien werden wie Unterverzeichnisse angesprochen. Dateien lassen 
  sich bequem in und aus gepackten Dateien kopieren, das entsprechende Pack-
  programm wird automatisch aufgerufen, ZIP ist fest eingebaut.
- Interne Entpacker fuer ARJ, LZH, TAR, GZ, CAB, RAR und ACE
- Startmenue (benutzerdefiniertes Menue).
- Konfigurierbarer Buttonbar.
- Kommandozeile zum einfachen und schnellen Starten von Programmen.
- Stark ausgebaute Suchfunktion mit Volltextsuche auch in gepackten Dateien.
- Eingebauter Dateibetrachter (F3), der Dateien beliebiger Groesse als Hex- oder
  Text darstellen kann, sowohl im ASCII- (DOS) als auch im ANSI- (Windows)
  Zeichensatz, zusaetzlich BMP-Dateien, andere Grafikdateien via Irfanview.
- PC-Direktverbindung via Parallelport
- Leistungsfähiges Umbenennwerkzeug
- Drucken von Verzeichnissen, Unterverzeichnissen und Verzeichnisbäumen
- Hintergrund-Transfermanager fuer laengere Dateioperationen

Die Shareware-Version von Total Commander ist in keiner Weise eingeschraenkt,
bis auf einen Startbildschirm, der bei jedem Programmstart erscheint und auf
die Registrierung hinweist.

Entfernen von Total Commander: Starten Sie wcuninst.exe !

Installation VON HAND: Sollte die Installation aus irgendeinem Grund fehl-
schlagen, so entpacken Sie die Datei INSTALL.CAB mit einem CAB-Entpacker
(z.B. CABVIEW von Microsoft) in ein leeres Verzeichnis, z.B. c:\totalcmd.
Verschieben Sie dann alle Dateien mit der Endung .pif in das Windows-
Verzeichnis! Dies bewirkt, dass die Packprogramme im Fenster ablaufen.
Die Dateien im Unterverzeichnis d\ müssen ins selbe Verzeichnis verschoben
werden wie totalcmd.exe.
