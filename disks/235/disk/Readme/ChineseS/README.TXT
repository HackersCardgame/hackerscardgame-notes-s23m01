   这片软盘包含重要的更新软件及在成品 CD 以后又增加的附加驱动器支持。当 CD 中
的安装程序也提示时, 请插入这片软盘。

   有些用户无法轻而易举地同时连接软盘驱动器和 CD-ROM 驱动器。例如，您有一台可
以使用软盘驱动器或 CD-ROM 驱动器的便携机，但是不能同时安装这两种驱动器。
如果您遇到这种问题，可以依照下列步骤：

1). 连接软盘驱动器。
2). 在硬件上创建一个临时文件夹。例如，c:\tempfile。
3). 插入这片软盘，并将里面所有内容复制到 c:\tempfile。
4). 断开软盘驱动器，然后连接CD-ROM 驱动器。
5). 插入 Adaptec CD 并遵循安装程序的指示。
6). 当提示“插入 Adaptec 更新软盘”时，单击“浏览...”按钮并键入

     c:\tempfile。
7). 单击[确定]。
8). 安装完成后，重新启动系统。如果您愿意的话可以删除 c:\tempfile文件
    夹中的文件。
