Diese Diskette enthält wichtige Software-Aktualisierungen und zusätzliche 
Treiber, die sich nicht auf der CD befinden. Legen Sie bei entsprechender 
Aufforderung durch das Setup-Programm diese Diskette in das Laufwerk ein.

Bei einigen Benutzern stellt die gleichzeitige Verwendung von Disketten- und 
CD-ROM-Laufwerk ein Problem dar, zum Beispiel bei einem Laptop-Computer, an 
den nicht beide Geräte gleichzeitig angeschlossen werden können. Gehen Sie in 
diesem Fall wie folgt vor:

1.)  Schließen Sie das Diskettenlaufwerk an.
2.)  Erstellen Sie einen temporären Ordner auf Ihrer Festplatte, zum Beispiel 
     c:\tempfile.
3.)  Legen Sie die Diskette in das Laufwerk ein, und kopieren Sie den gesamten
     Inhalt nach c:\tempfile.
4.)  Entfernen Sie das Diskettenlaufwerk, und schließen Sie das CD-ROM-Laufwerk
     an.
5.)  Legen Sie die Adaptec CD ein, und befolgen Sie die Anweisungen zur 
     Installation.
6.)  Klicken Sie bei der Aufforderung zum Einlegen der Aktualisierungsdiskette 
     auf "Durchsuchen...", und geben Sie "c:\tempfile" ein.
7.)  Klicken Sie auf "OK".
8.)  Nach Abschluß der Installation und dem Neustart des Systems können 
     Sie die Dateien im Ordner c:\tempfile wieder löschen.
