   這片軟碟片包含有重要的更新軟體以及在成品 CD 之後又增加的附加光碟機支援。
當 CD 中的安裝程式也提示時, 請插入這片軟碟片。

   有些使用者無法輕鬆的將軟碟式驅動器及 CD-ROM 驅動器同時連接。例如,您有一台可
以使用軟碟式驅動器或 CD-ROM 驅動器的可攜式電腦,但是不能同時安裝這兩種驅動器。
如果您遇到這種情形,可以遵循下列步驟:

1). 連接軟碟式驅動器。
2). 在硬碟上制作一個臨時資料夾。
3). 插入這片軟碟片, 並拷貝里面所有內容至 c:\tempfile。
4). 中斷軟碟式驅動器, 然後連接 CD-ROM 驅動器。
5). 插入 Adaptec CD 並遵循安裝程式的指示。
6). 當提示"插入 Adaptec 更新軟碟片"時, 按一下"瀏覽..."按鈕並鍵入
    c:\tempfile。
7). 按一下[確定]。
8). 安裝完成後, 重新啟動系統。如果您願意的話, 可以刪除 c:\tempfile 資料夾中
    的資料。
