このフロッピーディスクには、同梱されている CD-ROM が製造された後で追加された、
ソフトウェアの重要なアップデートと、ドライブサポートが含まれています。
CD-ROM 上のセットアッププログラムの指示に従って、このフロッピーを挿入してくだ
さい。

フロッピードライブとCD-ROM ドライブを同時に接続できない場合には、次のようにし
てください。例えば、フロッピードライブとCD-ROM ドライブを使用できるが、同時
には接続できないラップトップをお持ちの場合などです。
このようなユーザの方は、次の手順に従ってください:

1.)  フロッピードライブを接続します。
2.)  ハードディスクに一時フォルダ（例えば、c:\tempfile）を作成します。
3.)  本フロッピーを挿入して、すべての内容を c:\tempfile　にコピーします。
4.)  フロッピードライブを取り外し、CD-ROM ドライブを接続します。
5.)  Adaptec CD-ROM を挿入し、セットアップの指示に従います。
6.)  「Adaptec アップデートフロッピーディスク」を挿入するように指示されたとき、
     [参照...]ボタンをクリックし、「c:\tempfile」を入力します。
7.)  [OK] をクリックします。
8.)  インストールが完了し、システムを再起動した後で、c:\tempfile フォルダ内の
     ファイルを削除してもかまいません。
