Este disco flexible incluye importantes actualizaciones de software y apoyo 
adicional de unidades, que fueron agregados después de que se fabricó el CD. 
Introduzca este disco cuando se le indique en el programa de instalación en el CD.

Hay casos en que no se puede conectar fácilmente tanto una unidad de disco flexible 
como una de CD-ROM al mismo tiempo. Por ejemplo, quizá con su equipo portátil 
pueda utilizar tanto una unidad de disco flexible como una de CD-ROM, pero no 
tener ambas instaladas al mismo tiempo. Si tiene este problema, siga estos pasos:

1.)  Conecte la unidad de disco flexible.
2.)  Cree un archivo temporal en la unidad de disco duro. Por ejemplo, c:\archtemp.
3.)  Introduzca este disco y copie todo el contenido a c:\archtemp.
4.)  Quite la unidad de disco flexible y conecte la unidad de CD-ROM.
5.)  Introduzca el CD de Adaptec y siga las instrucciones de instalación.
6.)  Cuando se le indique que introduzca el disco de actualización de Adaptec, haga clic en
     el botón "Examinar..." y escriba c:\archtemp.
7.)  Haga clic en Aceptar.
8.)  Después de que se complete la instalación, y de que se reinicie el sistema, si lo desea, 
      puede borrar los archivos en la carpeta c\archtemp.
