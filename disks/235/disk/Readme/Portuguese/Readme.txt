Este disquete contém atualizações importantes ao software e suporte
adicional para unidades, que foram adicionados após a fabricação do
CD.  Insira este disquete quando solicitado pelo programa de instalação
existente no CD.


Existem alguns usuários que não podem com facilidade acionar uma
unidade de disco flexível e uma unidade de CD-ROM ao mesmo tempo.
Por exemplo, é possível que você tenha um laptop que pode ser usado
com uma unidade de disco flexível ou uma unidade de CD-ROM, mas 
que não pode ter ambas unidas instaladas ao mesmo tempo. Se você tiver
esta dificuldade, poderá seguir os seguintes passos:


1.)  Conecte sua unidade de disco flexível
2.)  Crie uma pasta temporária em sua unidade de disco rígido.  
     Por exemplo, c:\tempfile
3.)  Insira este disquete e copie todo o seu conteúdo para c:\tempfile
4.)  Remova o disquete e conecte a unidade de CD-ROM
5.)  Insira o CD da Adaptec e siga as instruções de instalação
6.)  Quando o programa solicitar que você insira seu "Disquete de Atualização 
     da Adaptec," clique no botão "Procurar..." e digite c:\tempfile
7.)  Clique em OK.
8.)  Após a instalação ter sido concluída, e o sistema tiver sido 
     reinicializado, você poderá excluir os arquivos da pasta c:\tempfile, 
     se assim o desejar.
